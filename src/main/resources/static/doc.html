Methods

/delete

This allows you remove user from database

/delete?username="username"

/switch

This change status of user to enabled if disabled and vice versa

/switch?username="username"

/add

This can add new user to database

/add?username="username"&password="password"&enabled="true/false"

/reset

This can be used to reset password for user

/reset?username="username"&password="new password"

/read

This can read specific user

/read?username="username"

/readAll

This reads all users

/readAll
